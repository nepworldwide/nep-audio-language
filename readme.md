# Audio Language


##Install
If you are starting a blank project from a clean composer.json
```angular2html

{
   "require":{
      "nepworldwide/nep-audio-language":"0.1"
   },
   "repositories":[
      {
         "type":"vcs",
         "url":"git@github.com:nepworldwide/nep-audio-language.git"
      }
   ]
}
```

Run
```angular2html
composer require nepworldwide/nep-audio-language
```
